Personal Jeopardy Game

Timeline: March-May 2020

Objective

Created an individual trivia game that is heavily inspired by Jeopardy. It keeps track of your score and money earned, constrains you to 30 seconds for each question, and logs separate attempts so that you can compete with others. There is also a rankings page that displays each attempt.

Outcomes & Contributions

Created entirely by myself.

Questions were tailored to my specific interests, such as my hobbies and memes I liked.

Technical Details & Skills

Language: Visual Basic .net
