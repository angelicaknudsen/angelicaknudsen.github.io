Objective

Program part of my undergraduate research in coastal acidification during summer of 2023. Designed to process and graph data from the Hanna, Manta +35, and YSI CastAway sensors in order to streamline research process. Using the Python library matplotlib, I converted files to .csv when needed, cleaned the data, and then graphed it on an interactive interface that allows users to cycle through through different parameters by simply clicking the window. Program is meant to be ran through command line interface.

Outcomes & Contributions

Cut about 5-10 minutes spent necessary for graphing data from each file, as some files had about 10+ parameters

Technical Details & Skills

Language: Python (matplotlib)
