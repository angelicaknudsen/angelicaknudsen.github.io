Objective

I wanted a website that acted as an engineering portfolio as well as a centralized place to showcase my many interests.

Outcomes & Contributions

I designed every part of this website using HTML, CSS, Nunjucks, Liquid, and Javascript. The website design takes inspiration from Adobe portfolio, but I wanted to make this website on my own as a fun challenge. The site is hosted on my GitHub.

For scalability, I use Eleventy. It is a static site generator, where I can easily design my own templates for different subpages. It is also nice being able to easily switch between using Nunjucks, Liquid, or HTML, depending on which is best suited for the different subpages.

Technical Details & Skills

Languages: HTML, CSS, and Javascript

Static Site Generator: Eleventy
